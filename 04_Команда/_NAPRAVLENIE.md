# 04 Команда

**Что это:** люди — база исполнителей, найм, оценка качества, условия работы.

**Цель сейчас (фаза 1):** к Q4 — **3 будние бригады**, работающие по стандартам из 02_Производство. Путь: объявления → собеседование → тест на реальном объекте (подработка) → постоянная работа. Каждый исполнитель — в komanda.xlsx с оценкой после каждого объекта.

**Цель потом:** найм прораба (год 2, фаза 2) — профиль готовится заранее в Найм/Профиль_прораба.md; дальше сметчик и бэк-офис (фаза 3).

**Метрика:**
- число рабочих бригад (план на конец года: 3)
- средняя оценка качества по komanda.xlsx (держать ≥ 4)
- текучка: сколько проверенных людей осталось через квартал

## Файлы
- `komanda.xlsx` — база исполнителей (специализация, ставки, оценка 1–5)
- `Найм/Профиль_бригады.md` — кого ищем в бригаду сейчас
- `Найм/Профиль_прораба.md` — заготовка на год 2
