# 05 Юридическое

**Что это:** правовая база — регистрация SIA, реестр būvkomersants, договоры, акты, субподряд.

**Цель сейчас (Q1 по плану / фактически — ближайшие недели):** **SIA зарегистрировано + внесение в būvkomersantu reģistrs** — без этого нет крупных заказчиков и тендеров DzĪB. До готовности SIA работаем как СД (saimnieciskās darbības veicējs).

**Цель потом:** полный комплект шаблонов (договор подряда, акт приёмки, договор субподряда с бригадами — критично к фазе 2, когда бригад станет много), страхование гражданской ответственности, при росте — юрист на аутсорсе.

**Метрика:**
- SIA: зарегистрировано да/нет
- Būvkomersants: в реестре да/нет
- Доля объектов, закрытых по полному комплекту документов (договор + акт): 100%

## Структура
- `SIA/` — учредительные документы, регистрация UR, документы būvkomersants
- `Шаблоны/` — договор подряда, акт приёмки, договор субподряда

## Ближайшие шаги
1. Название SIA, юр. адрес, уставный капитал, e-paraksts — собрать.
2. Регистрация в Uzņēmumu reģistrs, открытие счёта.
3. Заявление в būvkomersantu reģistrs (нужно SIA + ответственный специалист).
4. Шаблон договора подряда — показать юристу до первого крупного объекта.
