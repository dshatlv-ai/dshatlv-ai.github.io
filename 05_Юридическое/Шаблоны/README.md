# Шаблоны документов

Сюда складывать рабочие шаблоны (заполняются под объект):

- `Dogovor_podryada.docx` — договор подряда с заказчиком (DzĪB/частник) — **сделать до следующего объекта**, показать юристу
- `Akt_priemki.docx` — акт приёмки-передачи работ (суммы = смета + согласованные допработы)
- `Dogovor_subpodryada.docx` — договор с бригадой/мастером — критично, когда бригады станут постоянными

Смета — не здесь: генератор в 01_Продажи/Сметы_КП.

Минимум, который должен быть в договоре подряда: стороны и реквизиты, смета как приложение, сроки, порядок оплаты (аванс 40% / остаток по акту), допработы только письменно, гарантия и её границы, ответственность за скрытые дефекты основания (см. формулировки из смет Vārnu — «Piezīmes un nosacījumi»).
