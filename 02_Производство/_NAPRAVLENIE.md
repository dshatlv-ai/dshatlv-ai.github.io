# 02 Производство

**Что это:** исполнение объектов — от замера до подписанного акта. Бригады, закупки, качество, сроки.

**Цель сейчас (фаза 1):** каждый объект проходит **единый цикл**: замер → договор → закупки → работы → фото → акт. Ноль переделок: сдаём по чек-листу приёмки до того, как объект видит заказчик.

**Цель потом:** производство работает без меня — прораб (фаза 2) ведёт объекты по этим же стандартам; стандарты = инструкции для новых бригад.

**Метрика:**
- переделки: **0** (каждая переделка — запись в Решения.md с причиной)
- соблюдение сроков договора (сдано вовремя / всего)
- маржа по объекту не ниже плановой из сметы (сверка с 03_Финансы)

## Структура
- `Объекты/` — папка на каждый объект, копировать `_ШАБЛОН_объекта/`
- `Стандарты/Чек-лист_приёмки_объекта.md` — прогнать перед сдачей
- `Стандарты/Критерии_качества.md` — стяжка / плитка / покраска
