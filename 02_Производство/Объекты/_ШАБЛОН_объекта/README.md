# Шаблон папки объекта

Новый объект = копия этой папки с именем `ГГГГ-ММ_Адрес_кратко` (например `2026-07_Varnu_8_kapnu_telpa`).

Подпапки — по циклу объекта:

- `01_Замер/` — фото «до», размеры, заметки с осмотра
- `02_Договор/` — смета (из генератора), договор, переписка по согласованию
- `03_Закупки/` — список материалов, чеки, накладные
- `04_Фото/` — процесс и «после» (для портфолио и споров)
- `05_Акты/` — промежуточные и финальный акт, подтверждения оплат

Правила:
1. Смета — только генератором (01_Продажи/Сметы_КП), цены из справочника.
2. Расходы объекта сразу заносить в 03_Финансы/finances_2026.xlsx.
3. Перед сдачей — чек-лист приёмки (../Стандарты/).
4. После сдачи: фото «до/после» → портфолио, отзыв заказчика.
