# Чек-лист приёмки объекта (перед сдачей заказчику)

Прогонять самому (позже — прорабу) ДО звонка заказчику о сдаче. Цель — 0 переделок.

## Общее
- [ ] Все позиции сметы выполнены (пройтись по смете построчно)
- [ ] Допработы сверх сметы согласованы письменно и учтены в акте
- [ ] Мусор вывезен, объект убран (полы помыты, пыль снята)
- [ ] Соседние поверхности не пострадали (двери, перила, почтовые ящики)
- [ ] Фото «после» сняты (те же ракурсы, что «до»)

## По видам работ
- [ ] Стяжка / пол — по критериям качества (ровность, трещины, звук)
- [ ] Плитка — швы, уступы, пустоты (простукать), затирка
- [ ] Покраска — равномерность при дневном и искусственном свете
- [ ] Примыкания и углы — чисто, без наплывов герметика/краски

## Документы
- [ ] Акт приёмки подготовлен (суммы сходятся со сметой + допработы)
- [ ] Остаток к оплате посчитан, реквизиты в акте верные
- [ ] Гарантийные условия проговорены (что гарантия покрывает, что нет)

## После подписания
- [ ] Оплата остатка получена → отметить в 03_Финансы
- [ ] Отзыв заказчика (текст или хотя бы устное согласие на рекомендацию)
- [ ] Фото → портфолио; кейс — в материалы для обзвона
