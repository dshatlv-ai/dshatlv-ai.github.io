# 03 Финансы

**Что это:** деньги компании — учёт по объектам, маржа, подушка безопасности, план/факт.

**Цель сейчас (фаза 1):** **маржа 40–50%** на каждом объекте (контроль в finances_2026.xlsx сразу при составлении сметы и по факту закрытия); собрать **подушку на 2 объекта вперёд** (покрывает материалы + бригаду двух средних объектов, ориентир ~3–4 тыс €) — лист «Подушка».

**Цель потом:** финмодель компании — бюджет на прораба и офис (фаза 2), оборотный капитал под крупные объекты с поэтапной оплатой (фаза 3), инвестиционный капитал (фаза 4). Бухгалтерия SIA на аутсорсе.

**Метрика:**
- маржа по каждому закрытому объекту (цель ≥ 40%)
- размер подушки в € и в «объектах вперёд»
- оборот факт vs план квартала (сверка с 00_Управление/План_2026)

## Правила
1. Объект попадает в finances_2026.xlsx в день подписания договора.
2. Расходы заносятся по ходу, не «потом по памяти».
3. С каждого закрытого объекта — фиксированный % в подушку (решить и записать в Решения.md).
4. Первый объект внесён: Vārnu iela 8 — коридоры, маржа 50% — это планка.
