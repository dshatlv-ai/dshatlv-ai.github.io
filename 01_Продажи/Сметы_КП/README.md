# Генератор смет (Lokālā tāme)

Скрипт `generate_smeta.py` собирает смету `.xlsx` точно в формате образца
`Smeta_ieejas_zona_PASUTITAJS.xlsx`: шапка с реквизитами, секции работ с серыми
заголовками, колонки Nr / Nosaukums / Mērvienība / Daudzums / Cena / Summa /
Piezīmes, живые формулы `=D*E` и `=SUM(...)`, итоги по секциям, строка
`KOPĀ (PVN netiek piemērots)`, примечания и подписи внизу.

## Установка (один раз)

```bash
pip install openpyxl
```

## Запуск

```bash
python3 generate_smeta.py                # интерактивный режим
python3 generate_smeta.py --catalog      # показать справочник работ
python3 generate_smeta.py --demo         # тестовая смета для проверки
python3 generate_smeta.py --out Смeты    # сохранять готовые файлы в папку
```

## Интерактивный режим — пример сессии

```
Название объекта (2-я строка шапки): Ieejas zonas remonts — Vārnu iela 8
Адрес объекта: Vārnu iela 8, Rīga
Заказчик (строка целиком) [Pasūtītājs: DzīB "Vārnu iela 8, ..."]:   ← Enter = по умолчанию

> + GRĪDA UN KĀPNES          ← новая секция (серый заголовок)
> демонтаж пола; 11           ← позиция из справочника, 11 m²
> 2; 11                       ← можно по номеру из справочника (2 = стяжка)
> облицовка ступеней; 3       ← 3 п.м., поиск по части названия
> покраска; 20; 14            ← 20 m² по цене 14 EUR вместо каталожной
> Hidroizolācija; 5; 52       ← нет в справочнике → скрипт спросит единицу
>                                измерения и предложит сохранить в справочник
>                             ← пустая строка = закончить
Готово: Smeta_Ieejas_zonas_remonts_..._2026-07-08.xlsx
KOPĀ bez PVN: 1160.00 EUR
```

Формат строки позиции: `название-или-номер; количество[; цена за единицу]`.
Количество и цену можно писать с запятой (`11,5`). Поиск работает и по
русскому названию («стяжка»), и по латышскому («klons»), и по номеру.

## Справочник работ — rabota_catalog.xlsx

Лежит рядом со скриптом, создаётся автоматически при первом запуске со
стартовыми позициями (демонтаж пола, стяжка, плитка на пол и ступени,
покраска, шпаклёвка, грунтовка, вывоз мусора, контейнер, доставка).

Пополнять можно двумя способами:

1. **В Excel** — просто дописать строку: `Nr | Nosaukums (LV) | Название (RU) |
   Mērvienība | Cena EUR | Piezīme`. Колонка RU нужна только для поиска.
2. **Из скрипта** — ввести позицию, которой нет в справочнике: скрипт
   спросит детали и предложит сохранить.

**Стартовые цены — ориентировочные, проверь и поправь под свои!**

## Реквизиты и тексты

Исполнитель, заказчик по умолчанию, стандартные примечания и строки подписей
задаются константами в начале `generate_smeta.py` (блок «РЕКВИЗИТЫ И ТЕКСТЫ
ШАПКИ») — правятся в любом текстовом редакторе.
