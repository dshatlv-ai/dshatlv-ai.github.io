# 01 Продажи

**Что это:** приток денег — лиды, обзвон DzĪB, замеры, сметы и КП, договоры.

**Цель сейчас (фаза 1):** системный обзвон DzĪB **5–10 контактов каждую неделю** без пропусков; довести конверсию цепочки замер → смета → договор. Все лиды и их статусы — только в klienti.xlsx, ничего «в голове».

**Цель потом:** входящий поток (репутация, отзывы DzĪB, портфолио) + тендеры управляющих компаний; продажи крупных объектов 15–40 тыс €, дальше 50–200 тыс €.

**Метрика:**
- контактов DzĪB в неделю (план: 5–10)
- конверсия: звонок → замер → договор (считать раз в месяц по klienti.xlsx)
- сумма подписанных договоров за квартал

## Файлы
- `Лиды/klienti.xlsx` — база DzĪB и заявок, статусы, следующие шаги
- `Сметы_КП/` — генератор смет + справочник цен (см. README там)
- `Скрипты/Скрипт_звонка_DzIB.md` — сценарий холодного звонка
